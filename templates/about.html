关于 BeeLine Web Scheduling Console

平台简介

BeeLine是一个基于FastAPI的Python脚本调度运行平台，提供Web界面来管理Python脚本的执行和调度，特别适合需要定期执行脚本的场景。

核心价值

  • 集中化管理所有Python脚本
  • 可视化任务调度和监控
  • 简化脚本部署和执行流程
  • 提供完整的执行日志和错误追踪

功能特点

脚本管理

  • • 上传、编辑和删除 Python 脚本
  • • 语法高亮和代码格式化

任务调度

  • • 手动执行和定时执行
  • • 支持Cron表达式配置

包管理

  • • 查看已安装Python包
  • • 安装和卸载包
  • • 包版本管理

环境变量

  • • 脚本特定环境变量
  • • 环境变量分组
